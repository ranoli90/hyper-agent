HyperAgent Privacy Policy

Last updated: February 21, 2026

TL;DR: Your data stays on your device. We don't sell your data. API keys are stored locally in your browser. Chat history is saved locally and never sent to our servers.

1. Data We Collect

1.1 Data Stored Locally (On Your Device)

HyperAgent stores the following data locally in your browser using Chrome's extension storage:

  • API Key: Your OpenRouter/OpenAI API key (stored in plain text - protect your browser access)
  • Chat History: Your conversation history with the AI agent
  • Command History: Previous commands you've entered
  • Site Strategies: Learning data about successful actions on specific websites
  • Settings & Preferences: Extension configuration options
  • Scheduled Tasks: Any automation tasks you've set up
  • Session Snapshots: Saved agent states for resumption

1.2 Data Sent to Third Parties

When you use HyperAgent, the following data may be sent to third-party services:

  • LLM Provider (OpenRouter/OpenAI/Anthropic): Your commands and page context are sent to generate AI responses. This is governed by their respective privacy policies.
  • Stripe: If you purchase a subscription, payment processing data is handled by Stripe. We receive only subscription status and customer ID.

1.3 Data We Do NOT Collect

  • We do not collect personal identification information
  • We do not track your browsing history beyond what's needed for agent operation
  • We do not sell or share your data with advertisers
  • We do not have access to your API keys or chat history

2. How We Use Your Data

HyperAgent uses stored data exclusively for:

  • Executing your commands and automating browser tasks
  • Improving agent performance through learned strategies
  • Resuming previous sessions and tasks
  • Displaying your chat history
  • Managing your subscription (if applicable)

3. Data Security

  • API Keys: Stored locally using Chrome's extension storage. We recommend using API keys with usage limits. Never share your API key.
  • Local Storage: Data is protected by your browser's security model. Clear extension data to remove all stored information.
  • Transmission: All API calls use HTTPS encryption.

4. Data Retention

  • Chat history is retained until you clear it (/clear command) or uninstall the extension
  • Site strategies are retained indefinitely to improve performance
  • Session snapshots are retained until manually deleted
  • Subscription data is retained for billing purposes

5. Your Rights (GDPR/CCPA)

5.1 Right to Access

You can view all stored data using the /export command in HyperAgent.

5.2 Right to Delete

You can delete your data by:

  • Using Settings â†’ Reset All Settings (clears all local data)
  • Uninstalling the extension (removes all extension data)
  • Contacting us at privacy@hyperagent.ai for subscription data deletion

5.3 Right to Portability

Export your settings and chat history using /export and /export-chat commands.

6. Third-Party Services

6.1 LLM Providers

When you send commands, text and page context are sent to your configured LLM provider. Review their privacy policies:

  • OpenRouter Privacy Policy
  • OpenAI Privacy Policy
  • Anthropic Privacy Policy

6.2 Payment Processing

Stripe handles all payment processing. See Stripe's Privacy Policy.

7. Children's Privacy

HyperAgent is not intended for children under 13. We do not knowingly collect data from children under 13.

8. Changes to This Policy

We may update this policy periodically. Changes will be reflected in the "Last updated" date. Continued use of HyperAgent constitutes acceptance of changes.

9. Contact Us

For privacy-related questions or data deletion requests:

  • Email: privacy@hyperagent.ai
  • GitHub: github.com/ranoli90/hyper-agent/issues

10. Permissions Justification

HyperAgent requests the following browser permissions:

  • <all_urls>: Required to automate actions on any website you navigate to
  • tabs: To interact with browser tabs for automation
  • activeTab: To access the current tab when triggered
  • scripting: To inject automation scripts into pages
  • storage: To save your settings and chat history locally
  • unlimitedStorage: To store learning data and chat history
  • contextMenus: To provide right-click menu options
  • sidePanel: To display the agent interface
  • alarms: To schedule automated tasks
